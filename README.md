# Proposal Pack for Claude Code

A Claude Code skill pack that generates prospect-ready proposals for local marketers and agencies. Run audits, research prospects, and produce branded PDF proposals from the command line.

## What's Included

| Skill | What It Does |
|-------|-------------|
| `/setup` | Configure your agency name, services, brand colors, and pricing |
| `/brand-audit <url>` | Scan a website for voice, visual identity, SEO, and conversion signals |
| `/lighthouse-audit <url>` | Run a Lighthouse performance audit with scores and Core Web Vitals |
| `/prospect-research <name>` | Build a business dossier using web search |
| `/proposal <url>` | Chain all three audits into a branded HTML + PDF proposal |

## Quick Start

### Prerequisites

- [Claude Code](https://claude.ai/code) installed and authenticated
- Python 3.x
- Node.js (for Lighthouse)
- Chrome or Chromium (for Lighthouse headless)

### Install

```bash
git clone https://github.com/aisandler/brand-audit-v1.git
cd brand-audit-v1
pip install -r requirements.txt
```

### First Run

```bash
claude
```

Then inside Claude Code:

```
> /setup
```

Follow the prompts to configure your agency profile. This creates `config/agency-config.json` with your name, contact info, services, and default pricing template.

### Generate a Proposal

```
> /proposal https://example.com
```

This runs prospect research, a brand audit, and a Lighthouse audit on the target site, then assembles everything into a proposal with your agency branding and pricing. Output files land in `output/`.

## Individual Skills

Each skill works standalone:

```
> /brand-audit https://example.com
> /brand-audit https://example.com --save-json

> /lighthouse-audit https://example.com
> /lighthouse-audit https://example.com --save-json

> /prospect-research "Joe's Pizza Austin"
> /prospect-research dentists in Portland
```

Use `--save-json` to write structured results to `output/` for further processing.

## Pricing Templates

Four pricing structures ship out of the box in `config/pricing-templates/`:

- **Website Redesign** — Starter ($2,500) / Growth ($5,000) / Premium ($10,000)
- **Monthly Retainer** — Essentials ($997/mo) / Growth ($2,500/mo) / Scale ($5,000/mo)
- **One-Time Audit** — Deep Audit ($1,500)
- **Hybrid** — Launch ($4,000 + $750/mo) / Accelerate ($7,500 + $1,500/mo) / Dominate ($15,000 + $3,500/mo)

Customize prices and line items during `/setup` or edit the JSON files directly.

## How It Works

1. `/setup` interviews you once and saves your agency profile to `config/agency-config.json`
2. `/proposal` runs three analyses inline (prospect research, brand audit, Lighthouse)
3. Results are written as JSON to `output/`
4. The HTML proposal template is populated with findings and your agency branding
5. WeasyPrint converts the HTML to a print-ready PDF
6. All three output formats (MD, HTML, PDF) are saved to `output/`

## Limitations

- Brand audit is a surface scan using WebFetch (no JavaScript rendering)
- Lighthouse requires Chrome/Chromium installed on the system
- WeasyPrint PDF does not render Google Fonts (falls back to system fonts)
- Full proposal generation takes approximately 4-5 minutes
- Cannot analyze pages behind authentication or paywalls
